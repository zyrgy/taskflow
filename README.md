# TaskFlow

A clean, fast task management app built with React.

## Features
- Task name, owner, due date, priority, last updated, notes
- Add / edit / delete tasks
- Mark tasks complete
- Filter by status and priority
- Search by task name or owner
- Overdue detection
- Persists to localStorage

## Local development

```bash
npm install
npm start
```

## Deploy to Vercel via GitHub

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Import your GitHub repo
4. Vercel auto-detects Create React App — click **Deploy**
5. Done! Your app is live at `https://your-project.vercel.app`

No environment variables needed.
